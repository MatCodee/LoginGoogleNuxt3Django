### Instalación 🔧

_Una serie de ejemplos paso a paso que te dice lo que debes ejecutar para tener un entorno de desarrollo ejecutandose_

Frontend:
```
npm install
npm run dev
```

Backend:
```
virtualenv env
source env/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
